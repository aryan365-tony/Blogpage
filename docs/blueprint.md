# **App Name**: Kakehashi Hub

## Core Features:

- Bilingual Post Creation: Users can create posts with language metadata (English or Japanese) and request automatic translation. The AI tool enables side-by-side bilingual viewing.
- Language Switch: A prominent language switch button allows users to toggle the UI language between English and Japanese.
- Collaborative Editing: Users can comment on posts, suggest edits, and fork posts to propose collaborative changes, fostering cross-country exchange.
- User Profiles: Profiles display affiliations, research interests, and collaboration history, encouraging connections between Japanese and Indian researchers and students.
- Rich Media Editor: Markdown editor with WYSIWYG toolbar supports rich formatting including headings, code blocks, LaTeX, images, and file attachments, facilitating comprehensive research communication.
- Mentorship Matching: A mentorship matching system connects students and researchers based on topic/affiliation, enhancing the exchange of knowledge and experience.
- User Authentication: User roles management and signup via Google, ORCID, Email, or University SSO.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5), evoking a sense of trust, knowledge, and serenity. Inspired by the collaborative spirit and the pursuit of academic exchange, avoiding cultural cliches.
- Background color: Very light gray (#F0F2F5), providing a clean, distraction-free backdrop that's easy on the eyes and ensuring readability.
- Accent color: Vivid Violet (#9C27B0), adding a touch of dynamism and interest while remaining harmonious with the primary color, guiding user attention to key interactive elements.
- Body font: 'PT Sans', a humanist sans-serif suitable for body text. Headline font: 'Playfair', a modern serif with an elegant feel. Note: currently only Google Fonts are supported.
- Simple, clear icons for navigation and actions. Icons related to research, collaboration, and language, creating a cohesive visual language.
- Clean, modern layout with clear visual hierarchy, prioritizing usability and readability, and making important elements stand out.
- Subtle animations and transitions to enhance user experience, guiding users and providing feedback, creating a dynamic and engaging interface.
